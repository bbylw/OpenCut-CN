项目结构

清晰的模块化架构

apps/web/ — Next.js 主 Web 应用
src/components/ — UI 和编辑器组件
src/hooks/ — 自定义 React Hooks
src/lib/ — 工具函数和 API 逻辑
src/stores/ — 状态管理（Zustand 等）
src/types/ — TypeScript 类型定义

快速开始

几分钟内启动本地开发环境

Bun Docker （可选） Docker Compose （可选）

Docker 是可选的，但推荐用于运行本地数据库和 Redis。如果只想开发前端功能，可以跳过 Docker。

1

Fork 并克隆仓库

2

复制环境文件

# Unix/Linux/Mac

cp apps/web/.env.example apps/web/.env.local

# Windows PowerShell

Copy-Item apps/web/.env.example apps/web/.env.local

3

启动数据库和 Redis

docker compose up -d db redis serverless-redis-http

4

安装依赖并启动

bun install

bun dev:web

应用将在 http://localhost:3000 可访问

🐳 使用 Docker 自托管

使用 Docker 运行所有服务（包括应用的生产构建）

docker compose up -d

应用将在 http://localhost:3100 可访问
